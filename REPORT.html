01 — 产品概述

搜索 · 分析 · 对话

NEWSTAKER 是一款 AI 驱动的全球新闻智能平台。它不只是新闻聚合——而是提供搜索 → 浏览 → AI 深度分析 → 金融影响预测的完整闭环，让每一条新闻都成为可挖掘的数据节点。

产品拥有独特的 Paper Brutalism 视觉语言——米白底纹、深色线框、等宽字体、网格背景——在复古报纸质感与现代信息设计之间取得了精准平衡。中央的线框地球作为视觉锚点，赋予界面全球化的空间感知。

02 — 核心功能

五大核心能力

Search
全球新闻搜索
支持自然语言关键词搜索，Cmd/Ctrl+K 快捷聚焦。搜索结果实时映射到侧栏新闻列表，每条新闻附带分类标签、情绪分析、来源媒体和时间戳。支持中英文双语。
Chat
AI 新闻对话
长按新闻卡片进入 AI 对话模式。基于 GPT-4o 的多轮对话，可深入讨论新闻背景、影响分析、关联事件。对话面板居中滑入，支持无限轮追问。
Finance
金融数据 + 影响预测
左侧面板实时显示 NASDAQ / XAU / S&P500 指数。点击新闻情绪标签触发 AI 分析：预测 3 只看涨 + 3 只看跌股票及理由，点击个股可进入深度分析对话。
Mode
双模式引擎
默认 LLM 演示模式——AI 生成高度逼真的新闻数据。三击右上角状态灯切换到 newsdata.io 真实 API，获取实时新闻与原文链接。两种模式无缝切换。
Design
Paper Brutalism 视觉系统
完整的设计语言体系：米白网格底纹、深色线框边框、Archivo Black 标题字体、Space Mono 等宽数据字体、Inter 正文。7 大分类色彩编码（冲突红、政治蓝、经济绿、科技黄、健康紫、环境橙、社会灰）。所有交互面板遵循统一的视觉规范，中央线框地球作为全局视觉锚点，传递全球化信息感知。

搜索 · 新闻浏览 · 金融影响预测

输入关键词后，AI 或真实 API 返回全球相关新闻，右侧栏以时间线形式展示，每条新闻标注分类、情绪、来源和摘要。与此同时，点击新闻情绪标签即可触发 AI 金融影响预测——左侧面板实时展示 PREDICTED GAINERS（看涨）和 PREDICTED LOSERS（看跌），每只个股附带涨跌幅预估和 AI 分析理由，点击个股还可进入深度对话。

▲ 搜索 "AI chip technology" 后：右侧展示 TSMC、NVIDIA、Intel 等相关新闻；左侧 AI 自动预测受影响个股——PREDICTED GAINERS: TSM +5.4%、NVDA +2.8% / PREDICTED LOSERS: INTC -3.1% 等，每只股票附带分析理由

AI 对话分析

长按任意新闻卡片约 0.6 秒，进入与 GPT-4o 的深度对话模式。可追问新闻背景、潜在影响、相关事件，AI 基于新闻上下文给出专业分析。

▲ 对话模式：面板居中滑入，顶部显示新闻摘要和来源 URL，底部输入框支持多轮追问。AI 提供背景分析、影响评估、关联事件等深度回复

位置卡片与附近探索

点击地球上的事件标记，弹出位置卡片显示坐标、分类和情绪信息，可一键搜索附近新闻，从 3D 视图过渡到 Leaflet 2D 高精地图。

▲ 位置卡片：展示事件地理信息、新闻分类与情绪标签，SEARCH NEARBY 按钮触发附近新闻探索

2D 高精度地图 · 事件详情

点击 SEARCH NEARBY 后，界面从 3D 地球平滑过渡到 Leaflet 2D 高精度地图。地图上以脉冲圆点标记附近事件，点击任意标记后，左侧展开事件地理信息面板（EVENT LOCATION：城市、坐标、分类、情绪），右侧展开新闻摘要面板（NEWS BRIEF：标题、摘要、来源、时间）。三栏布局将地理空间与新闻内容完美融合。

▲ 2D 高精度地图模式：左侧事件位置信息（坐标、分类 CONFLICT、情绪 NEGATIVE），中央 Leaflet 高精地图 + 事件标记，右侧新闻摘要（标题、来源 REUTERS、时间）
03 — 应用场景

不止是看新闻

场景一：真实模式 — 新闻浏览与深度分析

切换到 LIVE 模式后，NEWSTAKER 接入 newsdata.io 真实数据源。AI 对每条新闻自动补充分类、情绪分析和地理定位，成为一个具有深度分析能力的新闻工具：

  • 新闻记者和编辑 — 追踪全球热点分布，搜索特定话题的国际报道，用 AI 对话快速梳理事件脉络
  • 投资分析师 — 利用 AI 即时评估新闻对个股的潜在影响（看涨/看跌预测），辅助投资决策
  • 企业情报团队 — 监控行业动态和全球区域风险，AI 对话深入分析供应链和市场影响
  • 教育与研究 — 帮助学生和研究者以多维视角（分类、情绪、地理、金融影响）理解时事新闻
真实模式的核心价值

将新闻从"平面阅读"升维到"搜索 + 情绪分析 + AI 对话 + 金融预测"的多维智能体验。每一条新闻不再是孤立的文本，而是一个有分类、有情绪、有市场关联的数据节点。

场景二：LLM 模式 — 游戏 UI 与艺术创作

在 LLM 演示模式下，AI 可以根据任意主题即时生成全球新闻分布。这种"AI 按需生成内容 + 精致 UI 呈现"的能力，为以下场景打开了想象空间：

  • 开放世界游戏内置新闻终端 — Paper Brutalism 的视觉风格天然契合赛博朋克 / 科幻 / 蒸汽朋克美学。AI 基于游戏世界观即时生成新闻流，作为沉浸式 UI 组件嵌入游戏
  • 交互式叙事与数据艺术 — 艺术家用关键词驱动 AI 生成特定主题的全球事件可视化，作为数据装置或交互纪录片的展示媒介
  • 策略 / 模拟类游戏仪表盘 — "新闻侧栏 + 金融面板 + 情报分析"的信息架构天然适配全球态势感知界面
  • 世界观构建工具 — 科幻 / 奇幻创作者输入关键词，AI 即时生成虚构世界的完整新闻事件网络，辅助世界观设计
设计哲学

Paper Brutalism 风格——米白背景、深色线框、等宽字体、网格底纹——既是复古的，也是未来的。这套视觉语言可以无缝嵌入从赛博朋克到蒸汽朋克的任意美学体系，作为下一代游戏和交互媒体的 UI 范式。

04 — 开发历程

从概念到产品

PHASE 01 — 前端原型
视觉系统与界面框架
使用 Variant 生成初始前端原型。确定 Paper Brutalism 设计语言（米白底纹 + 深色线框 + 等宽字体 + 网格背景），实现主界面三栏布局：左侧金融数据面板、中央线框地球装饰、右侧新闻列表。7 大新闻分类颜色体系。单 HTML 文件，纯前端 Mock 数据。
PHASE 02 — 后端集成
Node.js 服务器 + GPT-4o API
构建 Node.js 后端（server.js），接入 GPT-4o 作为 AI 引擎。实现搜索、趋势、附近新闻三个 API 路由。LLM 负责生成新闻数据（含坐标、分类、情绪评分）以及新闻深度分析和股票影响预测。
PHASE 03 — 真实数据接入
newsdata.io API + 双模式切换
初始方案尝试 GDELT 开放数据（免费、无 API Key），但 Node.js http 模块调用时持续超时（curl 正常），多次调试后放弃。最终集成 newsdata.io API，实现三击状态灯切换 Demo/Live 双模式。真实新闻经 LLM 二次处理（补充坐标、分类、情绪），保留原始 URL。
PHASE 04 — 交互深化
AI 对话 + 股票分析 + 附近探索
新增长按触发新闻 AI 对话（多轮追问）、情绪标签点击触发股票影响分析（3 涨 3 跌预测）、点击个股进入深度对话。实现 3D→2D 地图过渡的附近新闻探索。构建完整的"搜索 → 浏览 → 分析 → 对话"闭环。
05 — 技术架构

轻量级全栈架构

NEWSTAKER 采用极简技术栈——单文件前端 + 轻量 Node.js 后端，无构建工具、无框架依赖、无数据库。追求"打开即用"的部署体验。

数据流

Demo 模式： 搜索请求 → Node.js → GPT-4o 生成新闻 JSON（含坐标、分类、情绪）→ 前端渲染到侧栏 + 地球标记
Live 模式： 搜索请求 → Node.js → newsdata.io 获取真实新闻 → GPT-4o 补充分类、情绪、坐标 → 前端渲染
AI 对话： 用户提问 → Node.js 代理 → GPT-4o → 流式返回渲染
股票分析： 情绪标签点击 → GPT-4o 预测 3 涨 3 跌个股 → 渲染到金融面板

06 — 未来改进与计划

下一步路线图

优先级 功能 描述
■ HIGH 流式响应 (Streaming) AI 对话和新闻生成改为 SSE 流式输出，消除等待空白，提升交互感知速度
■ HIGH 多数据源聚合 接入 GDELT、GNews、MediaStack 等多个新闻 API，交叉验证去重，提供更全面的全球覆盖
■ HIGH 真实金融数据 接入 Yahoo Finance / Alpha Vantage API，替换模拟数据为实时行情，增强金融分析的可信度
■ MID 对话历史持久化 保存 AI 对话记录，支持历史对话回顾和继续追问
■ MID 新闻分类过滤 添加 7 大分类过滤按钮，支持多选过滤新闻列表和地球标记
■ MID 时间轴回放 添加时间维度滑块，回放过去 24h/7d/30d 的新闻事件演变，观察热点迁移
■ MID 新闻关联网络 可视化相关新闻事件之间的因果 / 影响链，构建事件图谱
■ LOW 用户系统 + 收藏 登录后可收藏新闻、保存对话历史、自定义关注区域和分类
■ LOW 移动端适配 响应式布局 + 触摸优化，适配手机端浏览体验
■ LOW 游戏 SDK 封装 将 AI 新闻生成和 UI 组件封装为可嵌入的 SDK，供游戏开发者集成为内置新闻终端
长期愿景

NEWSTAKER 的目标不只是一个新闻产品，而是一个"AI 新闻智能引擎"—— 既是记者的调查工具、分析师的决策助手，也可以是游戏世界中的沉浸式情报终端、数据艺术家的创作画布。 搜索是入口，AI 是大脑，新闻是脉搏。
