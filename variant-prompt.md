# NewStaker — Variant 前端生成 Prompt

> 以下 Prompt 可直接粘贴到 Variant 中使用。

---

请帮我创建一个名为 **"NewStaker 新闻拾取器"** 的交互式前端单页应用。这不是传统的平面地图，而是一个 **3D 地球仪** 形态的全球新闻+金融数据仪表盘。

---

## 一、核心交互：3D 地球仪（非平面地图）

### 地球仪渲染
- 使用 **Globe.GL**（基于 Three.js/WebGL 的 3D 地球组件，CDN 引入：`https://unpkg.com/globe.gl`）渲染一颗可交互的 3D 地球
- 地球贴图使用暗色风格底图（推荐 NASA Black Marble 夜景贴图或 CartoDB Dark Matter 风格）
- 地球表面叠加一层半透明的国界线多边形（hexed polygons 或 GeoJSON 边界）
- 添加一层薄薄的大气辉光层（atmosphere glow），颜色为科技蓝 `#00d4ff`

### 四元数风格的自由旋转交互
- **拖拽旋转**：鼠标左键拖拽地球，使用类似四元数（Quaternion）的旋转方式，不锁定任何轴，地球可以沿任意方向自由旋转，手感流畅无万向节锁死
- **惯性动画**：松手后地球带有惯性旋转衰减（momentum / inertia），慢慢停下
- **滚轮缩放**：鼠标滚轮控制缩放，支持从太空视角（看到整颗地球）一直平滑缩放到区域特写
- **双击聚焦**：双击某个国家/地区，摄像机平滑飞入（fly-to animation）该区域，并自动放大
- **自动旋转**：空闲时地球缓慢自转，用户交互时停止，闲置5秒后恢复自转

### 初始视角：聚焦中国
- **页面加载后，地球的初始视角对准中国**，摄像机的经纬度设置为 `{ lat: 35.86, lng: 104.19, altitude: 1.8 }`
- 中国区域在初始画面中处于视觉中心偏大的位置，给人一种"以中国为起点，向全球探索"的感觉
- 左下角提供一个"🏠 归位"按钮，点击后摄像机平滑飞回中国初始视角

## 二、新闻事件层

### 新闻气泡
- 在地球表面使用 Globe.GL 的 `pointsData` 或 `customLayer` 渲染新闻事件标记
- 每个新闻事件是一个 **发光柱体（bar）或脉冲光点（pulse ring）**，从地球表面向外延伸
- 柱体/光点的 **高度** 代表事件热度（来源数量越多越高）
- 柱体/光点的 **颜色** 按分类区分：
  - 🔴 冲突/战争：`#ff4444`
  - 🔵 政治：`#4488ff`
  - 🟢 经济/商业：`#44ff88`
  - 🟡 科技/科学：`#ffdd44`
  - 🟣 健康：`#bb66ff`
  - 🟠 环境：`#ff8844`
  - ⚪ 社会/文化：`#cccccc`
- 鼠标悬浮在光点上时，显示一个小 tooltip 展示新闻标题

### 新闻详情侧边栏
- 点击地球上的新闻标记，从 **右侧滑出** 一个毛玻璃半透明侧边栏（`backdrop-filter: blur(20px)`，背景 `rgba(10,14,23,0.85)`）
- 侧边栏内容包括：
  - 🏙️ **城市实景图片**：顶部展示一张该城市/地区当前季节的代表性风景照片（见第四部分）
  - 📰 **新闻列表**：该地区相关的新闻条目，每条显示：标题、摘要（2行截断）、来源媒体 logo、发布时间（相对时间如"2小时前"）、情感标签（正面/中性/负面用颜色区分）
  - 🔗 每条新闻可点击跳转原文链接
- 侧边栏可通过点击关闭按钮或点击地球空白处关闭（带滑出动画）

### 分类过滤
- 地球左侧竖向排列 7 个分类图标按钮（图标+文字标签），点击可切换显示/隐藏对应分类的新闻
- 支持多选，被选中的按钮高亮，未选中的灰暗
- 切换分类时地球上的光点有渐显/渐隐的过渡动画

## 三、金融数据面板

在页面 **左下角** 放置一个金融数据迷你面板（半透明毛玻璃卡片），实时展示关键金融指标：

### 数据项
1. **纳斯达克综合指数 (NASDAQ Composite / ^IXIC)**
   - 显示：当前点数、涨跌幅（百分比）、涨跌用红绿色区分
   - 迷你折线图（sparkline）：展示最近 7 天的走势缩略图
2. **伦敦金价 (XAU/USD)**
   - 显示：当前金价（美元/盎司）、涨跌幅
   - 迷你折线图：最近 7 天走势
3. **标普 500 (S&P 500 / ^GSPC)**（可选，作为补充）
   - 同上格式

### 数据来源（Mock）
- 由于是前端 demo，使用 **模拟数据** 生成逼真的金融数据
- 模拟数据包含：当前价格、日涨跌额、日涨跌幅（%）、7天历史价格数组（用于 sparkline）
- 金融面板每隔 5 秒模拟一次微小的价格波动动画（数字跳动效果，类似交易终端）
- 在面板底部小字标注 "数据仅供演示 · Demo Data Only"

### 视觉风格
- 面板宽度约 280px，圆角卡片
- 上涨用绿色 `#00e676`，下跌用红色 `#ff5252`（国际惯例配色）
- 数字使用等宽字体（monospace），价格变动时有数字翻滚/闪烁微动画
- Sparkline 迷你折线图使用 Canvas 或 SVG 绘制，线条细且带渐变填充

## 四、城市实景照片

### 功能说明
当用户点击某个新闻事件、打开侧边栏时，侧边栏顶部展示一张 **该城市/地区当季的代表性风景照片**。

### 实现方式
- 预置一个城市-照片映射表（JavaScript 对象），包含全球主要城市在不同季节的照片 URL
- 照片使用 **Unsplash Source** 的直链格式：`https://images.unsplash.com/photo-{PHOTO_ID}?w=600&h=300&fit=crop`
- 每个城市提供 4 张照片（春/夏/秋/冬），根据当前月份自动选择对应季节
- 季节判断逻辑：北半球 3-5月春、6-8月夏、9-11月秋、12-2月冬；南半球相反

### 预置城市照片列表（至少包含以下城市）
北京、上海、东京、伦敦、纽约、巴黎、悉尼、莫斯科、迪拜、开罗、里约热内卢、新德里、柏林、罗马、首尔、曼谷、伊斯坦布尔、内罗毕、墨西哥城、多伦多

### 照片展示效果
- 照片区域高度 180px，宽度撑满侧边栏
- 使用 `object-fit: cover` 裁切
- 照片底部叠加一层 **渐变遮罩**（从透明到深色），上面浮现城市名称（大字白色）和当前季节+月份（小字浅灰）
- 照片加载时显示一个骨架屏（skeleton placeholder）动画

## 五、整体布局与设计风格

### 页面结构
```
┌────────────────────────────────────────────────────┐
│  [Logo: NewStaker 🌍]          [🔍 搜索栏]  [区域选择]  │
│                                                      │
│  ┌──分类──┐                           ┌──侧边栏──┐  │
│  │ 🔴冲突 │                           │ 🖼️城市照片│  │
│  │ 🔵政治 │                           │ ────────  │  │
│  │ 🟢经济 │       🌍 3D 地球仪         │ 📰新闻1   │  │
│  │ 🟡科技 │      （占据全屏背景）       │ 📰新闻2   │  │
│  │ 🟣健康 │                           │ 📰新闻3   │  │
│  │ 🟠环境 │                           │  ...      │  │
│  │ ⚪社会 │                           └──────────┘  │
│  └────────┘                                          │
│  ┌──金融面板──┐                                      │
│  │ NASDAQ  ↑ │          [⏱️ 时间范围选择]             │
│  │ 金价 XAU ↓│          [🏠归位]                      │
│  └───────────┘                                       │
└────────────────────────────────────────────────────┘
```

### 设计语言
- **主题**：深色科技风（Dark Sci-Fi Dashboard）
- **背景色**：`#070b14`（极深蓝黑）
- **卡片背景**：`rgba(15, 20, 35, 0.75)` + `backdrop-filter: blur(20px)`（毛玻璃）
- **主色调**：霓虹蓝 `#00d4ff`（用于高亮、边框、图标）
- **强调色**：暖橙 `#ff6b35`（用于重要按钮、通知）
- **文字颜色**：主文字 `#e0e6f0`，次要文字 `#6b7a99`
- **字体**：标题使用 `'Orbitron', sans-serif`（科技感），正文使用 `'Inter', sans-serif`，数字使用 `'JetBrains Mono', monospace`
- **圆角**：卡片 12px，按钮 8px
- **所有浮动面板** 有 1px 的半透明蓝色发光边框 `border: 1px solid rgba(0, 212, 255, 0.15)`
- **阴影**：元素使用微弱的蓝色外发光 `box-shadow: 0 0 30px rgba(0, 212, 255, 0.1)`

### 动画
- 地球自转：默认 0.1°/frame 的自转速度
- 新闻光点：出现时从地面"升起"并伴随脉冲波纹动画
- 金融数字：价格变动时数字有 CSS transition 跳动效果
- 侧边栏：`transform: translateX(100%)` → `translateX(0)` 的滑入，带 `cubic-bezier(0.16, 1, 0.3, 1)` 缓动
- 页面初始加载：地球从远处飞近 → 旋转到中国视角，有一个 2 秒的 intro 动画

## 七、技术约束

- **纯前端实现**：单个 HTML 文件 + 内联 CSS/JS，或者 HTML + CSS + JS 各一个文件
- **外部依赖仅通过 CDN 引入**：
  - Globe.GL：`https://unpkg.com/globe.gl`
  - 字体：Google Fonts (Orbitron, Inter, JetBrains Mono)
- **不使用任何构建工具**（无 webpack/vite/npm）
- **不需要后端**，所有数据为前端 Mock
- 代码结构清晰，CSS 使用 CSS 变量统一管理主题
- 确保在 Chrome / Edge / Safari 最新版中正常运行
